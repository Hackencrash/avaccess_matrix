# AV Access HDMI Matrix (4KMX44-H2)

Home Assistant custom integration for controlling AV Access HDMI matrix switches.

## Features

- Control all 4 outputs via select entities
- Uses efficient `GET MP all` polling
- Event-driven updates after switching
- Device grouping in Home Assistant

## Installation (HACS)

1. Add this repo as a custom repository in HACS
2. Install "AV Access HDMI Matrix"
3. Restart Home Assistant
4. Add integration via UI

## Configuration

- Host: IP address of your matrix
- Port: 23 (default)

## Supported Devices

- AV Access 4KMX44-H2
